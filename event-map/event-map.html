Selected conflicts in the British Empire -- ca. 1500 to 1819

The colour scheme reflects different types of events which can be (de)selected in the filter. The categories are subject to interpretation and simplification of the insurgents' motives. Event descriptions in the labels have mainly been harvested from Wikidata / Wikipedia. Checking and updating the data is an on-going process. The current version of the geocoded data can be downloaded in GeoJSON format.
The MAGNIFYING GLASS permits searches for locations, the BINOCULARS select dates. The autofocus on relevant events only works for smaller map sections.
